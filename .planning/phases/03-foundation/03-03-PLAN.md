    Example invocation:
    ```
    node --env-file=.env.local -e "import('@neondatabase/serverless').then(({neon}) => neon(process.env.DATABASE_URL)('select 1 as ok, current_database() as db').then(r => console.log(JSON.stringify(r))).catch(e => { console.error(e.message); process.exit(1) }))"
    ```

    Capture the JSON output. The smoke test PASSES if the row returns `{ ok: 1, db: '<database name>' }`.

    If the smoke test fails with an authentication error, halt and report — likely the URL was copied incorrectly. If it fails with a network/TLS error, document and halt.
